# Portfolio Design Directions

This folder contains five distinct directions for Skyler's developer portfolio. Each option is designed to feel polished, modern, and professional while keeping the focus on projects, ideas, and career links.

## Quick comparison

| Proposal | Best for | Overall feel |
| --- | --- | --- |
| Constellation Storyline | A memorable personal brand and artful browsing | Narrative, calm, distinctive |
| Quiet Grid Showcase | Clear project-first presentation | Minimal, elegant, practical |
| Midnight Grid Atlas | A polished hybrid of clarity and atmosphere | Cinematic, structured, modern |
| Studio Casebook | Strong case studies and product thinking | Structured, editorial, confident |
| Signal Resume Hybrid | Fast recruiter scanning and credibility | Efficient, sharp, professional |

## Recommended use

- If Skyler wants the most distinctive and personal experience, start with **Constellation Storyline**.
- If clarity and easy maintenance matter most, choose **Quiet Grid Showcase**.
- If Skyler wants the clearest hybrid of usability and mood, **Midnight Grid Atlas** is the strongest blend.
- If the goal is to show depth of thinking behind projects, pick **Studio Casebook**.
- If the audience is heavily recruiter- or hiring-manager-focused, **Signal Resume Hybrid** is the strongest fit.
